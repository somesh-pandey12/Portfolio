# Foodverse 🍔

Foodverse is a full-stack food delivery web application built using the MERN stack. It provides a seamless user experience for browsing menus, managing a shopping cart, and securely ordering food online.

## 🚀 Features

- **User Authentication:** Secure user registration, login, and session persistence implemented using Passport.js.
- **Cart Management:** Dynamic shopping cart with real-time state updates for adding, removing, or modifying food items.
- **Database Integration:** Optimized MongoDB schemas with structured validation and data seeding for menus.
- **Responsive UI:** Clean and modern interface built using React and styled for all screen sizes.

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS / Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose Object Modeling)
- **Authentication:** Passport.js
